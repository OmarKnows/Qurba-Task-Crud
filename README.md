# Qurba-Task-Crud
This is my submission for Qurba's November 2022 backend interview task, wish me luck.


## Run Locally

Clone the project

```bash
  git clone https://github.com/OmarKnows/Qurba-Task-Crud.git
```

Go to the project directory

```bash
  cd Qurba-Task-Crud
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start:dev
```
